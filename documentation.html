Compared to What

Compared to What is an API that finds similar cities based on census data. To see the API in action, you can use Compared to Where, an interactive search tool.


Documentation

Find similar cities

Route: /get-similar
Variables:

  • place (required) - the name of the target city to find comparisons for. It must be the exact name of a U.S. census place.
  • fliter - the name of the filter applied to the search. This changes how census metrics are weighted to return different comparision cities. Defaults to "Smart", our recommended filter for general searches. The list of all filters can be found here.
  • n - the number of cities to return (max 100). Defaults to 10.

Returns a list of similar cities to the target place based on census data.

Example: comparedtowhat.azurewebsites.net/get-similar?place=Chicago+city,+Illinois&filter=Population&n=15

Get all places

Route: /all-places

Returns a list of all recognized census places.

Example: comparedtowhat.azurewebsites.net/all-places

Get all filters

Route: /all-filters

Returns a list of all available filters

Example: comparedtowhat.azurewebsites.net/all-filters

See all filters and their descriptions

Route: /filters

Shows a list of all available filters and their descriptions

Example: comparedtowhat.azurewebsites.net/filters

See a filter's description

Route: /filter-description
Variables:

  • fliter (required)- the name of the filter.

Returns a description of what metrics are important for the filter

Example: comparedtowhat.azurewebsites.net/filter-description?filter=Smart

See a filter's weights

Route: /filter-weights
Variables:

  • fliter (required)- the name of the filter.

Shows a table of the filter's weights

Example: comparedtowhat.azurewebsites.net/filter-weights?filter=Smart

Get a place's latitude and longitutde

Route: /latlong
Variables:

  • place (required) - the name of the target city to find comparisons for. It must be the exact name of a U.S. census place.

Returns a list containing the place's latitude and longitude.

Example: comparedtowhat.azurewebsites.net/latlong?place=Chicago+city,+Illinois
